This course will comprehensively the Github hosting service as a companion to the Git source control tool, which means no prior knowledge or experience is required. Students will emerge at the end with a very solid understanding and hands-on experience with Git and Github.

Course Organization

The course is divided into four major components:

  • Introduction and Setup
  • Learning Git Source Control
  • All About GitHub
  • Bonuses

Each one of the above components spans multiple sections in this course.
